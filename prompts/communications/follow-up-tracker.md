# Follow-Up Tracker

> Use to make sure nothing slips. Run after meetings or at end of day.

## Prompt

```
Based on [today's meetings / this conversation / this email thread], identify all follow-ups:

For each one, give me:
1. **What** — The specific action
2. **Who** — Who owns it (me or someone else)
3. **When** — Deadline or suggested timeline
4. **Draft** — If it's a message I need to send, draft it now

Then organize them:
- My follow-ups (things I need to do)
- Waiting on (things others owe me — I need to track these)

If anyone is overdue on something from my "Waiting on" list, draft a gentle nudge.
```

## When to Use

After every meeting. End of every day. Whenever the CEO says "don't let me forget to..."
