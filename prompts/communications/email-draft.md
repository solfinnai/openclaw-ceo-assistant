# Draft an Email

> Use when the CEO needs to send an email and wants a draft.

## Prompt

```
Draft an email:

- **To:** [Name / Role]
- **Subject:** [Topic]
- **Goal:** [What do I want the recipient to do after reading this?]
- **Tone:** [Warm / Formal / Direct / Grateful — default to my profile preference]
- **Context:** [Any background the assistant needs]
- **Length:** [Short (3-5 sentences) / Medium (1-2 paragraphs) / Detailed]

Write the email in my voice. Use my sign-off from my profile. No fluff.

If this is a follow-up, reference the prior conversation naturally.
If this is a cold outreach, keep it under 5 sentences.
```

## When to Use

Any email the CEO wants drafted — investor updates, candidate outreach, partner follow-ups, thank-you notes.
