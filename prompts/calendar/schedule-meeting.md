# Schedule a Meeting

> Use when the CEO says "set up a meeting with [person]."

## Prompt

```
Schedule a meeting:

- **With:** [Name(s)]
- **Purpose:** [What it's about]
- **Urgency:** [This week / Next week / Whenever works]
- **Duration:** [Default to my profile preferences if not specified]
- **Format:** [In-person / Video / Phone / No preference]

Do the following:
1. Check my calendar for available slots that respect my focus blocks and schedule preferences
2. Suggest 3 time options
3. Draft a short scheduling email I can send (or you send on my behalf)
4. If this meeting needs a briefing doc, flag that and I'll ask you to prep one
```

## When to Use

Anytime the CEO wants to book time with someone.
