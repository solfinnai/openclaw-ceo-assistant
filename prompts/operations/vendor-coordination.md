# Vendor Coordination

> Use when the CEO needs to manage a vendor, service provider, or contractor relationship.

## Prompt

```
I need help with a vendor:

- **Vendor:** [Name / Company]
- **Service:** [What they do for us]
- **Issue/Task:** [What needs to happen — e.g., "renew contract," "get a quote," "escalate a support issue," "cancel service"]
- **Context:** [Any background]
- **Urgency:** [Now / This week / Whenever]

Do the following:
1. Draft the email or message I need to send
2. If this is a negotiation, suggest my position and any leverage I have
3. If this is a cancellation, check for any contractual obligations or notice periods I should know about
4. Track the next step and follow-up date
```

## When to Use

Contract renewals, service issues, new vendor onboarding, or any external operations work.
