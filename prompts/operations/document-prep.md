# Document Prep

> Use when the CEO needs a document drafted, organized, or reviewed.

## Prompt

```
I need a document:

- **Type:** [Board deck outline / One-pager / Memo / Agenda / Brief / Talking points]
- **Audience:** [Board / Investors / Team / External partner]
- **Purpose:** [What is this document supposed to accomplish?]
- **Key points to cover:** [Bullet points or rough notes]
- **Tone:** [Formal / Conversational / Data-heavy]
- **Length:** [One page / 2-3 pages / As needed]
- **Deadline:** [When do I need this?]

Draft it. Use my voice. Structure it so the audience can scan it quickly. Lead with the most important thing.

If you need more information to write a good draft, ask me — don't guess.
```

## When to Use

Board meeting prep, investor decks, internal memos, or any time the CEO needs something written that isn't an email.
