# Expense Report

> Use to compile expenses after a trip or at end of month.

## Prompt

```
Help me compile my expenses:

Here are my receipts/transactions:
[Paste or describe transactions]

For each expense:
1. **Date**
2. **Vendor**
3. **Amount**
4. **Category** — [Travel / Meals / Software / Office / Other]
5. **Business purpose** — One sentence explaining why this was a business expense
6. **Receipt status** — Do I have a receipt? Flag any missing ones.

Then give me:
- Total by category
- Overall total
- Any missing receipts I need to track down
- Formatted for [my expense tool from my profile]
```

## When to Use

After trips, end of month, or whenever expenses pile up.
