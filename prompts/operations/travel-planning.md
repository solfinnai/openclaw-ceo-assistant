# Travel Planning

> Use when the CEO has a trip to plan.

## Prompt

```
Plan my trip:

- **Destination:** [City/Location]
- **Dates:** [Travel dates]
- **Purpose:** [Meeting / Conference / Team offsite / Personal]
- **Budget:** [Range or "standard"]
- **Key meetings/events:** [Anything already locked in]

Put together:
1. **Flights** — Options that match my travel preferences (from my profile). Include times and airlines.
2. **Hotel** — 2-3 options near my meetings. Match my preferences.
3. **Ground transport** — Car service, rental, or ride-share plan.
4. **Itinerary** — Day-by-day schedule combining meetings + travel logistics.
5. **Packing/prep reminders** — Anything I need to bring, sign, or review before the trip.
6. **Dinner/meals** — If I'm taking anyone out, suggest restaurants near the hotel.

Don't book anything — just give me the plan so I can approve it.
```

## When to Use

Any time the CEO travels for work.
