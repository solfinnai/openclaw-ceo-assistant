# Friday Closeout Workflow

> Run every Friday afternoon. Closes the week clean.

## Steps

1. **Weekly review** — Use `weekly-review.md` prompt.
2. **Update priorities** — Based on the week's outcomes, suggest edits to `current-priorities.md`.
3. **Clear follow-ups** — Use `follow-up-tracker.md`. Mark done items, surface overdue ones.
4. **Prep for Monday** — Flag anything that needs weekend attention or Monday-morning action.
5. **Draft investor/stakeholder update** — If it's end of month, use `investor-update.md`.

## Output Format

```
WEEK IN REVIEW — [DATE RANGE]

COMPLETED:
- [What got done]

OPEN ITEMS:
- [What's still in progress]

COMMITMENTS MADE:
- [New things the CEO committed to this week]

OVERDUE:
- [Anything slipping]

NEXT WEEK PREP:
- [What to focus on Monday]
```
