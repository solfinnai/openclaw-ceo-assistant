# Meeting Debrief Workflow

> Run immediately after any important meeting.

## Prompt

```
I just finished a meeting:

- **With:** [Who]
- **Topic:** [What it was about]
- **My notes:** [Paste raw notes, voice memo transcript, or just talk through it]

Do the following:
1. **Summary** — Clean up my notes into a clear 3-5 bullet summary
2. **Decisions made** — What was agreed on?
3. **Action items** — Who owes what, by when?
4. **Follow-up messages** — Draft any emails or messages I need to send as a result
5. **Update priorities** — Does anything in `current-priorities.md` need to change?
6. **Flag risks** — Anything concerning that came up?

If I need to loop anyone in, draft that message too.
```

## When to Use

After investor calls, board meetings, key hire interviews, partner conversations — any meeting with consequences.
