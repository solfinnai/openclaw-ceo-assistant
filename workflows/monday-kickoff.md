# Monday Kickoff Workflow

> Run every Monday morning. Gets the CEO aligned for the week.

## Steps

1. **Pull this week's calendar** — Use `daily-briefing.md` prompt for today, then summarize the full week.
2. **Review priorities** — Read `current-priorities.md`. Flag anything that's off-track.
3. **Check follow-ups** — Use `follow-up-tracker.md`. Are there overdue items? Draft nudges.
4. **Prep for today's meetings** — Use `meeting-prep.md` for any meetings that need it.
5. **Deliver the briefing** — Combine everything into one clean message.

## Output Format

```
WEEK OF [DATE]

TOP 3 THIS WEEK:
1. [Priority]
2. [Priority]
3. [Priority]

TODAY'S SCHEDULE:
[Briefing]

OVERDUE FOLLOW-UPS:
[List or "All clear"]

PREP NEEDED:
[Meetings that need briefing docs]

SUGGESTED ACTIONS:
[2-3 things to do first]
```
